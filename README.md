# TEAM-Net: MEG Emotion Recognition

Task-adaptive framework for cross-subject emotion recognition using MEG signals with valence–arousal modeling and LOSO evaluation.

---

## Overview

This repository contains the implementation of **TEAM-Net**, a task-specific deep learning framework for emotion recognition from MEG signals.

- **Dataset:** OpenNeuro ds007640  
- **Task:** Emotion recognition (valence & arousal)  
- **Evaluation:** Leave-One-Subject-Out (LOSO)  
- **Models:**
  - CNN–BiLSTM (Arousal)
  - Multi-scale CNN (Valence)
